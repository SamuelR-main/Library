Numbers

Integer Type:

byte, short, int, long

Floating Point Type:

float, double

byte

data type can store whole number from: -128 to 127 = 2^8

short

data type can store whole number from: -32768 to 32767 = 2^16

int

data type can store whole number from: -2147483648 to 2147483647 = 2^32

long

data type can store whole number from: -9223372036854775808 to 9223372036854775807 = 2^64

float

data type can store whole fractional number "end with f"

double

data type can store whole fractional number "end with d"

boolean

data type can store whole from "true" or "false"

char

data type can store a single character, sigle quotes "a" or "C", can use ASCII Values

String

data type is used to store a sequence of caracters (text), double quotes

Java Type Casting

Widering Casting (Automatically)

converting a smaller type to a larger type size:

byte -> short -> char -> int -> long -> float -> double

Norrowing Casting (Manually)

Converting a larger type to a smaller size type:

double -> float -> long -> int -> char -> short -> byte

Widening Casting

int myInt = 9;
double myDouble = myInt;

Narrowing Casting

double myDollar = 9.78d;
int myInt = (int) myDouble;
